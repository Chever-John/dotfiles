# alacritty

这是一个很不错的工具。

我希望使用 alacritty + tmux + zsh + nvim（AstroNvim）来提高我的工作效率。

目前我的 alacritty keybindings 支持如下：

```shell
command+r  # 清屏，清理掉历史命令行的显示
command+w  # 隐藏，默认是直接quit了，这里改成隐藏
conmand+t  #新开窗口，假如需要第二个终端窗口
command+shfit+w #关闭当前窗口
command+delete #删除一行
command+f #搜索关键字
command+← #跳到行首
conmand+→ #跳到行尾
```
